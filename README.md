# 📈 Financial Data Analysis of Tesla and GameStop using Python

## 📌 Project Overview

This project analyzes the historical stock prices and quarterly revenue of Tesla and GameStop using Python.

Historical stock price data is collected using the **yfinance API**, while quarterly revenue data is extracted through **web scraping** using Requests and BeautifulSoup. The project demonstrates an end-to-end data analysis workflow, including data collection, preprocessing, visualization, and interpretation.

---

## 🎯 Project Objectives

- Retrieve historical stock price data using the yfinance API
- Extract quarterly revenue data using web scraping
- Clean and preprocess financial datasets
- Compare stock price trends with revenue performance
- Visualize financial insights using Python

2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Open the notebook in Jupyter Notebook or JupyterLab.
4. Run the notebook cells sequentially.
